OpenClaw 和本地模型 Ollama

Posted 2026-02-26 · Words 809

年初开始 OpenClaw 话题度只增不减，使用本地模型就可以零成本体验。

1. 安装本地模型，如果想用云端API可忽略

安装Ollama: curl -fsSL https://ollama.com/install.sh | sh

下载模型: ollama pull qwen3:4b

测试模型: ollama run qwen3:4b

2. 安装OpenClaw

安装OpenClaw: curl -fsSL https://openclaw.ai/install.sh | bash

安装向导: openclaw onboard --install-daemon

  • I understand this is powerful and inherently risky. Continue? -> Yes
  • Onboarding mode -> QuickStart
  • Config handling -> Update values
  • Model/auth provider -> Skip for now
  • Filter models by provider -> ollama
  • Default model -> ollama/qwen3:4b
  • Select channel (QuickStart) -> Skip for now
  • Configure skills now? (recommended) -> No
  • Enable hooks? -> Skip for now
  • Gateway service already installed -> Restart
  • How do you want to hatch your bot? -> Hatch in TUI

到这一步，你的本地 AI 助手就已经跑起来了。
