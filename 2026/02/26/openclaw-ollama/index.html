OpenClaw 和 本地模型 Ollama
2026-02-26

1. 安装本地模型，如果想用云端API可忽略

安装Ollama: curl -fsSL https://ollama.com/install.sh | sh

下载模型: ollama pull qwen3:4b

测试模型: ollama run qwen3:4b

2. 安装OpenClaw

安装OpenClaw: curl -fsSL https://openclaw.ai/install.sh | bash

安装向导: openclaw onboard --install-daemon

  • I understand this is powerful and inherently risky. Continue? -> Yes

  • Default model -> ollama/qwen3:4b
